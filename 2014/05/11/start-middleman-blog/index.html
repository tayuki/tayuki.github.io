middlemanでブログはじめました。

wordpressのセキュリティ意識するのが面倒なので、移行先ブログエンジンをさがしていました。途中octopressで構築していたけど、カスタマイズが思うようにいかないし、使っててしっくりこない..。 結局middlemanでblogを再構築することにしました。

rbenvはすでに導入ずみなので、$ gem install middleman ですぐ使える状態に。

追加したgemは以下のとおり。

gem 'middleman-deploy'
gem "middleman-google-analytics"

レイアウト等を一から書くのがしんどいので、GhostのCasparってテーマをmiddlemanのテンプレート化したものを使わしてもらってます。
