01 Jun 2014 on virtualbox FreeBSD Windows

ヘッドレスなFreeBSD上にvirtualboxでWindows8.1環境を構築

$ portsnap fetch
$ portsnap update

した後に、

$ cd /usr/ports/emulators/virtualbox-ose
$ make BATCH=yes install clean

でインストール

現在でバージョンは以下のとおり。

% VBoxManage -v
4.3.12_OSEr93733

※4.3でないとWindows8.1の64bit版がインストールできないらしい…。
